# Comando While

O comando while é a estrutura mais simples de repetição da linguagem C. Ele também pode ser chamado de laço ou loop de repetição.

```C
while(expressão lógica){
    instruções no bloco
}
```

```C
comandos de inicialização

while(expressão lógica){
    /*corpo do while*/
    instruções no bloco

    atualização
```
